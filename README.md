# bag
## node

    nodemon  npm install -g nodemon
    nodemon是一种工具，通过在检测到目录中的文件更改时自动重新启动节点应用程序来帮助开发基于node.js的应用程序。
    
    mocha    npm install -g mocha
    mocha是一种测试工具，node和brower都可以用
    
    chai     npm install chai --save-dev
    chai是断言工具
    
    
    nrm      npm i nrm -g
    nrm查看下载源
    
    cheerio  npm i cheerio -D
    在node里面类似于使用jquery
